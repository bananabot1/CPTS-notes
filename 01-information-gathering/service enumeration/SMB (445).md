**Overview:**
- SMB is a client-server protocol for sharing files, directories, and network resources (printers, interfaces) over a network. Uses TCP with a three-way handshake before establishing a session.
- Access to shares is governed by ACLs defined at the share level, which are independent from local filesystem permissions on the server. A misconfigured share can expose files the server-side ACLs would otherwise restrict.
- Samba is the Unix implementation of SMB, using the CIFS dialect. Older NetBIOS-based Samba connections use TCP ports 137, 138, and 139. CIFS and modern SMB operate exclusively over TCP port 445.
- SMB 1 (CIFS) is outdated and considered insecure but may still be present in legacy environments. SMB 2 and SMB 3 are preferred in modern infrastructure.
- Samba v3 can join an Active Directory domain as a member. Samba v4 can act as a full AD domain controller, making it a high-value target in mixed Linux/Windows environments.

## Dangerous Settings

|Setting|Description|
|---|---|
|`browseable = yes`|Shares are listed and visible to clients|
|`read only = no`|File creation and modification not restricted|
|`writable = yes`|Users can create and modify files|
|`guest ok = yes`|Connection allowed without credentials|
|`enable privileges = yes`|SID-assigned privileges are honored|
|`create mask = 0777`|Newly created files get full permissions|
|`directory mask = 0777`|Newly created directories get full permissions|
|`logon script = script.sh`|Script executed on user login|
|`magic script = script.sh`|Script executed when the script file is closed|
|`magic output = script.out`|Output destination for the magic script|

## Enumeration

```
smbclient -N -L //<target>
```

List available shares on the target using a null session. `-N` skips authentication, `-L` requests the share list.

```
smbmap -H <target>
```

Enumerate shares and their permission levels (read/write/none) for the current or anonymous user.

```
crackmapexec smb <target> --shares -u '' -p ''
```

Enumerate shares using explicit null session credentials.

```
rpcclient -U "" <target>
```

Open an RPC session for manual SMB interaction. Useful for user and domain enumeration beyond what share-listing tools expose.

|RPC Query|Description|
|---|---|
|`srvinfo`|Server information|
|`enumdomains`|All domains deployed in the network|
|`querydominfo`|Domain, server, and user info for deployed domains|
|`netshareenumall`|All available shares|
|`netsharegetinfo <share>`|Details about a specific share|
|`enumdomusers`|All domain users|
|`queryuser <RID>`|Details about a specific user by RID|

```
samrdump.py <target>
```

Enumerate users via the SAMR protocol using Impacket. Dumps usernames and RIDs without requiring credentials if null sessions are allowed.
