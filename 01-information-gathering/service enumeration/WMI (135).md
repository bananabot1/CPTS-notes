**Overview:**
- WMI is Microsoft's implementation of CIM/WBEM for Windows, providing read and write access to nearly all system settings. It is the primary interface for remote administration of Windows systems, covering both PCs and servers.
- Accessible via PowerShell, VBScript, and WMIC. Not a single program but a collection of programs and databases (repositories).
- Communication is initiated over TCP port 135. After the connection is established, traffic is moved to a randomly assigned high port.
---
## Connection

```
wmiexec.py <user>:"<password>"@<target> "<command>"
```

Execute a command on the remote host via WMI using Impacket's `wmiexec.py`. Returns command output directly to the attacker's terminal without dropping an interactive shell on the target.
